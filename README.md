# Ubuntu VM Setup Project

This project contains step-by-step instructions and resources for setting up an Ubuntu virtual machine on Windows using Oracle VirtualBox.

## Directory Structure

- **/docs**: This folder contains the most current and up-to-date documentation files. Refer to these files for the latest instructions and checklists.
- **Other files in the root**: These are legacy or earlier versions of the documentation. They are kept for reference but may be removed in the future.
- **keyword_tag_toy.md**: An example file demonstrating how to use keywords and tags in a markdown file.
- **uac.md**: Boilerplate content for User Account Control (UAC) dialogs, intended to be copied and pasted into other markdown files as needed.

---

For the best experience, always use the files in the /docs folder.
