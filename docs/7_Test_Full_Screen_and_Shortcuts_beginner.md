# Ubuntu VM Setup (Beginner Checklist)

# 7. Test Full Screen and Shortcuts (Beginner)

1. Try Host+F (Right Ctrl+F) to enter full screen mode in your VM.
2. The VM window should resize with no black margins.
3. Test clipboard copy/paste and drag-and-drop between your Windows PC and the VM.

If something doesn't work, check that Guest Additions are installed or ask for help.
