# Ubuntu VM Setup (Expert Checklist)

# 5. Install Ubuntu (Expert)

**Note:** Installation steps may change over time. If you notice small differences, follow the prompts. For major changes, update the checklist.

1. Launch VirtualBox.
2. Start the Ubuntu VM.
3. Begin Ubuntu installation:
	- Select language, keyboard layout, connect to internet if needed.
	- Choose **Interactive installation**.
	- Select default apps.
	- Skip proprietary software.
	- Select **Erase disk and install Ubuntu** (only affects VM).
	- Create account (name, computer name, username, password).
	- Set timezone.
	- Review and install.
4. Restart VM when prompted.
5. If asked to remove installation medium:
	- Devices > Optical Drives > Uncheck ISO.
	- Force Unmount if needed.
	- Press Enter in VM to finish reboot.

You should now see a log in screen for your Log in name.
