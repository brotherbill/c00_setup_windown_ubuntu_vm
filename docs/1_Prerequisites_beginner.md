# Ubuntu VM Setup (Beginner Checklist)

# 1. Prerequisites (Beginner)

This step ensures your computer is ready for setting up an Ubuntu virtual machine.

1. Make sure you are using a Windows 11 PC.
2. Confirm you have administrator rights (you may need to enter a password when installing software).
3. Check that you have at least 30GB of free disk space on your hard drive.

If you are unsure how to check any of these, ask for help or see the detailed instructions in the main guide.
