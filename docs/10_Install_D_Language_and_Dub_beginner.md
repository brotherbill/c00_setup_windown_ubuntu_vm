# Ubuntu VM Setup (Beginner Checklist)

# 10. Install D Language and Dub (Beginner)

1. Open a terminal in Ubuntu.
2. Run:
   ```bash
   sudo apt install -y curl
   curl -fsS https://dlang.org/install.sh | bash -s dmd
   ```
3. Follow the script's instructions to add D to your PATH (usually: `source ~/dlang/dmd-*/activate`).

If you need help, see the main guide or ask for assistance.
