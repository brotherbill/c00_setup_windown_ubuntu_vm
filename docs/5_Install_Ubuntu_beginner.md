# Ubuntu VM Setup (Beginner Checklist)

# 5. Install Ubuntu (Beginner)

1. Start the VM.
2. Select the Ubuntu ISO if prompted.
3. Follow the installer prompts: choose language, keyboard, and timezone.
4. Select **Erase disk and install Ubuntu** (this only affects the VM).
5. Create a username and password you will remember.
6. Finish the installation and reboot the VM.
7. Remove or detach the ISO if prompted.

If you get stuck, note the step and error, and ask for help.
