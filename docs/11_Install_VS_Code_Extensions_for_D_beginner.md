# Ubuntu VM Setup (Beginner Checklist)

# 11. Install VS Code Extensions for D (Beginner)

1. Open VS Code in Ubuntu.
2. Press Ctrl+Shift+X to open the Extensions view.
3. Search for and install:
   - code-d (D language support)
   - Native Debug (for GDB integration)
4. Alternatively, run in terminal:
   ```bash
   code --install-extension webfreak.code-d
   code --install-extension webfreak.debug
   ```

These extensions help you write and debug D programs in VS Code.
