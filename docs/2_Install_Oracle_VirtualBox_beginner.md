# Ubuntu VM Setup (Beginner Checklist)

# 2. Install Oracle VirtualBox (Beginner)

This step will help you install the VirtualBox software needed to run your Ubuntu virtual machine.

1. Open your web browser and visit: https://www.virtualbox.org/
   - This is the official website for VirtualBox, a free program that lets you run other operating systems inside your Windows PC.
2. Click the **Get Started Download** button.
   - This will take you to the downloads page. If you don’t see it, look for a button or link that says “Download VirtualBox.”
3. Click **Windows hosts** to download the installer.
   - “Windows hosts” means the installer is for Windows computers. The file will start downloading automatically.
4. Open the downloaded file (usually in your Downloads folder).
   - If you’re not sure where it went, check your browser’s downloads list or look in your Downloads folder.
5. When installing an application, Windows may display a User Access Control (UAC) dialog asking:

   “Do you want to allow this app to make changes to your device?”

   - On most personal Windows 11 computers, you can simply click **Yes** to continue.
   - If you are on a corporate or school device and cannot proceed, contact your tech support team for assistance.

6. Accept all default options during installation.
   - You can just click **Next** or **OK** at each step. The default settings are fine for most users.
7. Approve any prompts about network or USB drivers.
   - VirtualBox may ask to install extra drivers. Click **Install** or **Allow** if prompted. These help your VM work better.

If you have trouble, don’t worry! Ask for help or see the main guide for troubleshooting tips. It’s normal to have questions or run into small issues during installation.
