# 4. Build First D App (Beginner)

## Overview
Step-by-step guide to create and run your first D app.

### Steps
1. Open Terminal in your projects folder.
2. Run:
   ```sh
   ./create_d_project my_next_app
   ```
   (Replace `my_next_app` with your project name.)
3. Open the folder in VS Code:
   ```sh
   code my_next_app
   ```
4. Build and run:
   ```sh
   dub run
   ```
5. To debug, use the launch configuration in VS Code.
