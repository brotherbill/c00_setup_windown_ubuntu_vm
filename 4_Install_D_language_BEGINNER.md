# 3. Install D Language (Beginner)

## Overview
Beginner-friendly steps to install the D language compiler and tools.

### Steps
1. Open Terminal (Ctrl+Alt+T).
2. Run:
   ```sh
   sudo apt install dmd dub
   ```
3. Check installation:
   ```sh
   dmd --version
   dub --version
   ```
