# 2. Install VS Code (Beginner)

## Overview
Detailed steps for beginners to install Visual Studio Code on Ubuntu.

### Steps
1. Go to https://code.visualstudio.com/Download and download the .deb file.
2. Open the Downloads folder, right-click the .deb file, and select **Open with Software Install**.
3. Click **Install** and enter your password if prompted.
4. Launch VS Code from the app menu or run:
   ```sh
   code
   ```
5. Open Extensions (Ctrl+Shift+X), search for "C/C++" by Microsoft, and install it.
